# GoREST API Testing

API Tests Using Postman and Newman

### Requests:

- Get Users
- Get User by
- Create a new User
- Update a user
- Delete User
- Get Todo
- Get ToDo by Status
- Get Comment
- Negative scenarios

### Bonus:

- [x] Add a reporter to view run information.

### How to run the project

```console
npm run positive-test
```
or
```console
npm run negative-test
```

**Note:** Don't forget to install the node packages (``npm i``)
